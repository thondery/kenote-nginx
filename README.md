# kenote-nginx
nginx for vhost

## Install

```bash
npm i -g kenote-nginx
```

## Commands

```bash
  Usage: node-nginx [command] [options]


  Options:

    -V, --version  output the version number
    -h, --help     output usage information


  Commands:

    conf [path]  set nginx conf path...
    add          add nginx conf
    list         list nginx confs
    show         show nginx conf path
```
